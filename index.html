📊 Weekend Volume Trends  · Sat – Sun – Mon ·
Decline-Adjusted
Each day's gas sales are mathematically declined forward to Apr 12, 2026 using a 7.7% annual decline rate. This removes the natural production decline from the data so all weeks are on a level playing field — allowing a fair apples-to-apples comparison across time. The dashed avg line is based on 2025 weeks only (excluding the 2 lowest) so outlier events (e.g. freeze-offs) don't skew the baseline.
Actual
Raw reported gas sales with no adjustments. This is what was physically sold to the pipeline each day. Actual volumes trend lower over time as wells naturally decline, so older weeks will generally look higher than recent weeks without decline adjustment. The dashed avg line is based on 2025 weeks only (excluding the 2 lowest) so outlier events (e.g. freeze-offs) don't skew the baseline.
